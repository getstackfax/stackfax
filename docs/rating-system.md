# Stackfax Rating System

Stackfax uses a 1–5 star rating system to make Ai stacks easy to understand, compare, and improve.

The star rating is not just about raw power. A strong stack should match the user's actual use case, budget, safety needs, privacy level, and technical skill.

## ⭐ 1-Star Stack

A risky or poorly matched stack.

Common signs:

- Expensive models used for simple tasks
- No clear use case
- No approval gates
- Unclear permissions
- High privacy risk
- Unnecessary hardware purchases
- No cost controls
- Agents can touch important systems too easily

Verdict:

Do not scale this stack yet. Simplify it, reduce access, and define the use case first.

## ⭐⭐ 2-Star Stack

A usable but messy stack.

Common signs:

- Some useful tools or models are present
- Cost control is weak
- Model routing is unclear
- Safety rules are incomplete
- Hardware may be overkill
- The user may not understand what the agents can access

Verdict:

This stack can be tested, but it needs cleanup before real business, financial, or customer-facing use.

## ⭐⭐⭐ 3-Star Stack

A solid starter stack.

Common signs:

- Use case is mostly clear
- Basic safety rules exist
- Hardware choice is reasonable
- Cost risk is manageable
- Some approval gates are present
- Good enough for learning and controlled testing

Verdict:

Good starting point. Improve model routing, documentation, and approval gates before scaling.

## ⭐⭐⭐⭐ 4-Star Stack

A strong stack.

Common signs:

- Clear use case fit
- Good model routing
- Reasonable cost controls
- Strong approval gates
- Privacy risks are understood
- Hardware matches the workload
- Human review exists for important actions
- Setup is explainable to the user

Verdict:

This stack is ready for serious use with proper monitoring and discipline.

## ⭐⭐⭐⭐⭐ 5-Star Stack

An excellent stack.

Common signs:

- Cost-aware
- Use-case matched
- Well-routed
- Well-documented
- Safe by default
- Clear permission boundaries
- Strong human approval rules
- Hardware and hosting are justified
- Scalable without becoming reckless
- Easy to audit, explain, and improve

Verdict:

This is a high-quality stack. It is not just powerful — it is controlled, efficient, and matched to the job.

## Rating Principle

A 5-star stack is not always the biggest or most expensive stack.

A 5-star stack is the stack that best fits the job.

## Core Stackfax Rule

Cheap model drafts.  
Strong model decides.  
Human approves anything touching customers, money, inventory, credentials, or production systems.
